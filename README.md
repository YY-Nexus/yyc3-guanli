# YanYu Cloud Sharing E-center

## 项目简介

YanYu Cloud Sharing E-center 是一个现代化的企业智能管理系统，为企业提供全方位的数据分析、组织管理、绩效激励等解决方案。系统采用模块化设计，支持多种业务场景，帮助企业实现数字化转型和智能化管理。

## 🌟 核心特色

- **智能数据分析**: 多源数据整合，实时可视化分析
- **全面组织管理**: 人力资源全生命周期管理
- **科学绩效激励**: 目标导向的考核激励体系
- **优质客户服务**: 全渠道客户关系管理
- **高效流程管控**: 业务流程自动化优化
- **创新驱动发展**: 创新项目孵化支持
- **持续效率优化**: 运营效率提升方案
- **多元服务体系**: 个性化服务定制

## 🛠 技术架构

### 前端技术栈
- **框架**: Next.js 14 (App Router)
- **UI库**: shadcn/ui + Tailwind CSS
- **状态管理**: React Hooks
- **图标库**: Lucide React
- **类型检查**: TypeScript

### 设计系统
- **主题色系**: 
  - 数据中心: Emerald (绿色)
  - 组织管理: Amber (琥珀色)
  - 绩效激励: Amber (琥珀色)
  - 客户服务: Blue (蓝色)
  - 其他模块: Purple (紫色)
- **交互效果**: 立体悬停、平滑动画
- **响应式设计**: 移动端友好

## 🚀 功能模块

### 1. 数据中心
- **数据采集整合**: 多源数据接入、清洗规范、ETL流程管理
- **可视化分析**: 图表看板动态生成、多维钻取、实时监控
- **决策模型应用**: 智能算法支撑、场景化决策、预测分析
- **数据安全审计**: 权限管控、日志追溯、合规检查
- **数据质量管理**: 数据质量评估、异常检测、清洗规则
- **数据资产目录**: 元数据管理、血缘关系、资产盘点

### 2. 组织管理
- **组织架构设计**: 层级梳理、岗位配置、职责划分
- **人才招聘体系**: 渠道管理、简历智能筛选、面试流程
- **员工培训发展**: 课程体系、成长路径追踪、技能评估
- **人力成本核算**: 薪资社保、预算管控、成本分析
- **员工关系管理**: 员工满意度、离职分析、团队建设
- **考勤管理**: 打卡统计、请假审批、排班管理

### 3. 绩效激励
- **目标拆解与对齐**: 战略分解、部门个人指标、OKR管理
- **考核流程管理**: 周期设置、评分规则、进度跟踪
- **激励策略配置**: 奖金晋升、荣誉方案、积分体系
- **绩效数据分析**: 结果复盘、改进建议、趋势预测
- **360度评估**: 多维度评价、同事互评、上下级评估

### 4. 客户服务
- **客户信息管理**: 档案建立、标签体系、画像生成
- **服务流程设计**: 售前咨询、售后响应、投诉处理
- **满意度调研分析**: 问卷配置、结果统计、改进举措
- **客户分层运营**: VIP服务、沉睡客户唤醒、生命周期管理

### 5. 创新管理
- **创新提案征集**: 渠道搭建、创意筛选、评审流程
- **项目孵化支持**: 资源匹配、进度管控、里程碑管理
- **创新成果评估**: 价值测算、推广路径、ROI分析
- **创新文化建设**: 案例分享、激励活动、创新氛围
- **知识产权管理**: 专利申请、商标注册、版权保护
- **外部合作**: 产学研合作、技术引进、联合创新

### 6. 效率优化
- **效率诊断工具**: 流程耗时、资源占用分析、瓶颈识别
- **自动化方案落地**: 重复性工作替代、工具集成、RPA部署
- **跨部门协同优化**: 沟通机制、协作流程、信息共享
- **效率指标监控**: 实时看板、预警设置、KPI追踪
- **工作负载均衡**: 任务分配、资源调度、能力匹配
- **时间管理**: 时间追踪、效率分析、工作计划

### 7. 多元服务
- **专属服务定制**: 按需配置、个性化方案、定制开发
- **服务渠道管理**: 线上线下、触点优化、全渠道整合
- **服务质量监督**: 质检标准、问题闭环、持续改进
- **跨领域服务拓展**: 新场景挖掘、合作对接、生态建设
- **服务标准化**: SOP制定、培训体系、认证管理
- **服务创新**: 新服务开发、用户体验优化、技术应用

## 🎨 设计规范

### 视觉设计
- **主题色系**: 每个模块都有独特的主题色，保持视觉一致性
- **卡片设计**: 统一的左侧4px主题色边框，增强模块识别度

- **悬停效果**: 所有交互元素都有立体浮起效果

### 交互设计
- **导航系统**: 侧边栏导航，支持收缩和展开
- **面包屑**: 顶部显示当前位置和模块信息
- **响应式**: 完美适配桌面端和移动端
- **加载状态**: 所有异步操作都有loading状态

## 🚀 快速开始

### 环境要求
- Node.js 18.0+
- npm 或 yarn

### 安装依赖
\`\`\`bash
npm install
# 或
yarn install
\`\`\`

### 启动开发服务器
\`\`\`bash
npm run dev
# 或
yarn dev
\`\`\`

### 构建生产版本
\`\`\`bash
npm run build
# 或
yarn build
\`\`\`

### 启动生产服务器
\`\`\`bash
npm start
# 或
yarn start
\`\`\`
